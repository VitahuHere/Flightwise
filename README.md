# Flightwise
## Prerequisites
- Node.js with npm
- Python 3.10 or above
- Poetry for managing packages
## Running
1. Install packages
Frontend:
```bash
cd Frontend && npm install
```
Backend:
```bash
cd src && poetry install
```
2. Running
Frontend:
```bash
cd Frontend && npm run dev
```
Backend:
```bash
cd src && fastapi run main.py
```
